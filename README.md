# Task Manager

Приложение для управления повседневными делами.

Проект состоит из 2 частей:
- ## Сервер
    Серверная часть приложения, хранит информацию о пользователях и задачах в базе данных, устанавливает логику работы с данными и предоставляет REST API для клиентской части.

    Стек: С++, Boost, CMake, PostgreSQL, Flyway, Docker.
    
    Запуск:
    ```powershell
    cd server
    docker compose up
    cd ..
    ```
    Выключение:
    ```powershell
    cd server
    docker compose down
    cd ..
    ```
    
- ## Клиент
    Графический интерфейс для взаимодействия с сервером.
    Стек: C++, Qt, CMake.
    В Releases находится exe файл для запуска gui в windows.
